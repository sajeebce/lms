## â“ Common Questions

### **Q: What is Texril?**
A: A powerful rich text editor built with TipTap, Next.js, and React. Can be used standalone or embedded in websites.

### **Q: Do I need to install anything?**
A: Yes, you need Node.js (v18+). Download from https://nodejs.org/

### **Q: How do I run it?**
A: Run `start.bat` (Windows) or `./start.sh` (Linux/Mac). Or manually: `npm install && npm run dev`

### **Q: Where do I see the editor?**
A: Open http://localhost:3000 in your browser after running the server.

### **Q: Can I use this in my project?**
A: Yes! It's MIT licensed. Copy the entire `texril` folder anywhere.

### **Q: What packages are included?**
A: 57 packages total. See [DEPENDENCIES.md](./DEPENDENCIES.md) for the complete list.

### **Q: Can I deploy this?**
A: Yes! Deploy to Vercel, Netlify, or any Node.js hosting.

### **Q: Is this related to the LMS project?**
A: It's a standalone snapshot. Changes here don't affect the main LMS.
